# Frontend レビュー評価UI変更 報告書

## 変更日
2026-03-16

## 変更概要
レビュー評価UIを数値入力から3択ボタン（観た/良かった/最高）に変更。
数値はユーザーに見せず、ラベルのみ表示。

## 変更ファイル

### src/lib/rating.js（新規）
- `RATING_OPTIONS`: 評価選択肢の定義 `[{value:3, label:'観た'}, ...]`
- `ratingLabel(value)`: 数値→ラベル変換関数

### src/views/WorkDetailView.vue
- 投稿フォーム: `<input type="number">` → 3択トグルボタン（観た/良かった/最高）
- レビュー一覧: 数値表示 → `badge-amber` でラベル表示
- 平均値セクション追加: 「観劇メモ評価 4.3 / 12件のメモ」形式
- 空状態: 「まだ感想はありません」+「最初の観劇メモを残す」ボタン

### src/views/MyPageView.vue
- 感想タブ: 星アイコン連打表示 → `badge-amber` でラベル表示
- `IconStar` のインポートを削除

## UI仕様
- 評価ボタンはトグル式（もう一度押すと解除）
- 選択中はアンバー系の背景色でハイライト
- ユーザーに数値(3,4,5)は一切見せない
- レビューサイト的な星UIは使わない
